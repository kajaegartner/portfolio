About me

Hello, I'm Kaja! Wellcome to my first portfolio web page!

After earning Bachelor's degree from European Studies, I decided to continue education in the field of Social Informatics, where I became acquainted with the concepts of user experience (UX), accessibility, and usability testing. In October 2020 I decided it is time to learn more about UX so i took part in Interaction Design Foundation's UX courses. However, I felt the need to know more about UX from web developer's point of view.

I started learning HTML and CSS basics at the beginning of December 2020. You are looking at the page I was able to create ater one month of self-learning and completing W3Schools HTML Tutorial, LinkedIn CSS Essentials Course and FreeCodeCamp Responsive Web Design Course.

Even though, I was only interested at the basics at the beggining, playing around with HTML and CSS quickly started to feel like a form of art (and I love art and creating!), so I really started to enjoy it. I'm looking forward to learn more about front-end web development day by day. Hopefully, some day I will have enough knowledge to be able to become full-time front-end web developer or UX designer.
